# Gavin Loam｜知識腐植層

Gavin Loam 是 Gavin Shared-Root Ecosystem 的知識腐植層。它承接從服務、生活、學習與思考中累積下來的觀點、方法、案例與未完成材料。這些內容經過整理與沉澱後，可能被不同服務、系統或出版內容重新吸收，也可能暫時留在其中，成為未來生長的養分。

Gavin Loam 不等同正式規格，也不代表內容已被驗證或採用。

## 可能內容

- 原始觀察
- 對話整理
- 跨領域想法
- 未成熟框架
- 研究筆記
- 失敗紀錄
- 歷史版本
- 尚未決定用途的內容材料

## 內容生命週期

```text
Raw Observation
原始事件、靈感或對話
        ↓
Gavin Loam
整理、沉澱、保留尚未定稿的材料
        ↓
人工審核與驗證
        ↓
依用途分流
├─ Internal Knowledge Base｜正式內部規格
├─ Professional Forest｜被服務或系統吸收
├─ Gavin Books｜完整出版作品
└─ Public Content｜文章、圖卡、影片或公開工具
```

不得把 Gavin Loam 中的內容自動視為現行規則。
