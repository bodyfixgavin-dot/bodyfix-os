# Web Presentation｜公開網站呈現規格

本文件只建立呈現規格，不建立頁面、路由或 UI。

## Homepage｜首頁

- 首頁只呈現三座品牌林：BodyFix、Chart Navigator、Space Guide。
- 首頁主句：**「不同的專業，生長在同一片知識土壤。」**
- 首頁不呈現完整內部架構圖，不顯示 Supabase、API、Schema、資料表、migration 等內部技術名詞。
- BodyFix、Chart Navigator、Space Guide 可呈現為平行專業入口，但不得互相吸收或混成單一服務。

## Future `/ecosystem` Page｜未來完整架構頁

完整架構放在 `/ecosystem` 的未來頁面規格中，採三幕垂直敘事：

1. **Canopy & Forests**：對外品牌林與服務入口。
2. **Gavin Loam**：知識腐植、方法沉澱與材料代謝層。
3. **Shared Root Network**：帳號、資料、權限、付款與報告等底層能力。

手機版採垂直卡片流，不使用大型橫向組織圖。

## Visual Direction｜視覺方向

- 禁止卡通樹木、具象綠葉、蘑菇與桌遊風格。
- 品牌林使用暖白、深藍與精品編輯排版。
- Gavin Loam 使用紙張、土壤與沉積紋理。
- Shared Root Network 使用深海軍藍、細金根線與抽象節點。

## Scope Guardrails｜本次範圍限制

- 本次只建立 Markdown 呈現規格。
- 不建立 `/ecosystem` 頁面。
- 不修改首頁、路由、元件、API、Schema、設定或營運功能。
