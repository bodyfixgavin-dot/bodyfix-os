# Shared Root Network｜地下共根網絡

Shared Root Network 負責不同專業森林之間的底層連接與資料傳輸。它不是前台品牌、不是單一服務，也不是無限制資料互通的授權。

可能包含：

- CRM
- 使用者識別
- 登入
- 預約
- 付款
- AI Router
- 共用資料規格
- 權限與紀錄交換

## 文件角色

本文件只定義 Shared Root Network 的角色與邊界，不建立或修改任何技術實作。

## 邊界

- Shared Root Network 可以讓不同森林共用底層能力。
- Shared Root Network 不會讓 BodyFix、Chart Navigator、Space Guide 在前台混成同一項服務。
- Shared Root Network 不等於所有資料自動互通。
- 跨森林資料使用仍需符合 consent、permission、data classification 與最小必要原則。
