# Operations & Decision Layer｜營運與決策層

Operations & Decision Layer 負責觀察營運狀態、資源配置與決策。它可以包含不同專業森林各自的營運觀測工具，但不預設所有森林共用同一個中央控制台。

## BodyFix Pulse

BodyFix Pulse 是 BodyFix 的營運與決策觀測中心。

目前服務範圍以 BodyFix 為主，包括：

- 每日營收
- 客戶與回訪
- 預約狀態
- 工作負荷
- 每週決策
- 該撐、該修、該停、該轉的判斷

## 現階段限制

不要在現階段把 BodyFix Pulse 定義成整個生態所有品牌的唯一中央控制台。

未來 Chart Navigator、Space Guide 或其他專業森林是否共用 Pulse，需另行評估：

- 需要哪些營運指標？
- 是否與 BodyFix 指標同質？
- 是否涉及不同資料敏感度？
- 是否需要獨立 dashboard？
- 是否需要跨森林 aggregation？
