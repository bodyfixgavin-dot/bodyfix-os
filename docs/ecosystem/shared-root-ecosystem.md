# Gavin Shared-Root Ecosystem

Gavin Shared-Root Ecosystem 是最高層級的跨領域生態架構。它承載 BodyFix、Chart Navigator、Space Guide，以及未來可能新增的專業領域。

它是一套世界觀與架構名稱，不是單一前台產品、服務品牌、後台系統或付款入口。

## Professional Forests｜專業森林

目前包含：

- BodyFix
- Chart Navigator
- Space Guide

各專業森林必須維持：

- 獨立定位
- 清楚的客戶入口
- 明確服務邊界
- 可獨立理解的品牌語言

不同森林可以共用地下系統，但不可在前台混成同一項服務。

## Service Trees｜服務之樹

Service Trees 是各專業森林內可被使用、購買或持續培育的服務項目。

範例：

- BodyFix：筋膜鏈整理、骨盆核心整理、教練課
- Chart Navigator：紫微、吠陀占星、塔羅與決策工具
- Space Guide：租屋前判讀、入住後空間整理與方位工具

## 前台邊界

- BodyFix 的前台語言應聚焦身體狀態管理、筋膜鏈整理、張力判讀、骨盆核心整理與動作整合。
- Chart Navigator 的前台語言應聚焦命盤、占星、塔羅與決策整理工具。
- Space Guide 的前台語言應聚焦租屋、入住後空間整理、方位與空間工具。

三者可以在共根生態內互相轉介，但不得用其中一個森林的專業語言取代另一個森林的判讀。

## 與其他層的關係

現行 canonical 敘事採用三層垂直關係，避免把品牌林、知識層、底層能力與出版結果誤認為同一層級：

```text
Gavin Shared-Root Ecosystem
└─ Canopy & Forests｜對外品牌林
   ├─ BodyFix
   ├─ Chart Navigator
   └─ Space Guide
      ↓
   Gavin Loam｜知識腐植與方法代謝層
      ↓
   Shared Root Network｜帳號、資料、權限、付款與報告等底層能力
```

Operations & Decision Layer、Internal Knowledge Base 與 Published Works 是治理、知識定稿與出版功能，不作為首頁對外品牌林。歷史文件中曾並列多層角色的版本，保留為演進證據；目前有效決策見 [`architecture-decisions.md`](./architecture-decisions.md)，未決問題見 [`conflicts.md`](./conflicts.md)。
