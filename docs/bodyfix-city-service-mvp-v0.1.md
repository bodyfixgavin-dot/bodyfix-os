# BodyFix 城市服務日 MVP v0.1

## 1. 目標

建立一套可在 BodyFix OS 中運作的「城市服務日」系統，先完成：

1. 城市需求登記
2. 成本試算
3. 成行判斷
4. 訂金狀態
5. 同行折抵控制
6. LINE 城市場次入口

第一版先做資料結構、後台骨架與手動流程，不急著做宣傳海報、Share Target Picker 或全自動推薦。

## 2. 服務定位

- BodyFix 1 對 1 教練課
- 筋膜整理
- 運動按摩
- 塔羅快問
- 紫微結構分析
- 三個月深度整合方案

正式教練課名稱：**BodyFix 1 對 1 教練課**

方法名稱：**動作整合訓練**

## 3. 城市服務日命名

- 對外名稱：城市服務日
- LINE 選單：城市場次
- 內部後台：跨城排程

第一版城市：

- 桃園
- 新竹
- 宜蘭
- 台中
- 台南
- 高雄
- 花蓮
- 台東

花蓮與台東分開計算。

## 4. 台北排程規則

- 六張犁：日常基地
- 西門：西區集中據點
- 國父紀念館：東區完整服務據點
- 松山：教練課錨點，可做運動按摩，無筋膜床

核心規則：

> 六張犁是基地，教練課是當日錨點；錨點形成後，其他服務往同區或鄰近據點集中。

系統只能提供調度建議，不可自動更改客戶預約。

## 5. 城市需求登記

客戶填寫：

- 姓名或稱呼
- LINE 使用者識別
- 手機號碼
- 希望城市
- 想預約的服務
- 平日／週末
- 白天／晚上
- 可接受日期範圍
- 是否可前往鄰近城市
- 是否希望 Gavin 主動聯絡
- 意願程度

意願分級：

- 關注
- 有興趣
- 明確需求
- 待付款預約

需求登記不等於正式預約，也不在此階段收款。

## 6. 單人指定到訪

客戶可提出：

- 指定城市
- 希望日期
- 可接受日期範圍
- 服務項目
- 希望服務時間
- 是否可提供合適空間
- 是否有朋友一同預約

公開頁面先不寫死價格。

內部參考值：

- 宜蘭：NT$6,000 起
- 花蓮：NT$10,000 起
- 台東：NT$15,000 起

## 7. 成本試算

直接成本：

- 往返交通
- 當地交通
- 住宿
- 場地租借
- 餐食與雜支
- 耗材
- 金流手續費
- 設備或行李運送
- 額外場地延長費
- 同行折抵
- 風險準備金

時間成本：

- 往返交通時間
- 場地準備與收尾
- 不可利用空檔
- 住宿占用時間
- 台北可能失去的接案收入
- Gavin 希望保留的最低淨收入

公式：

**場次淨貢獻**

已確認營收  
－交通  
－住宿  
－場地  
－當地交通  
－金流  
－耗材  
－其他直接成本  
－同行折抵

**場次淨貢獻時薪**

場次淨貢獻  
÷  
交通、準備、服務與不可利用空檔總時間

## 8. 成行狀態

- DRAFT
- COLLECTING
- NEAR_THRESHOLD
- BOOKING_READY
- CONFIRMED
- FULL
- COMPLETED
- CANCELLED

成行必須同時看：

- 已確認營收
- 場次淨貢獻
- 場地是否可用
- 時段是否能連續安排
- 訂金是否達保障門檻
- Gavin 是否人工確認

## 9. 訂金流程

1. 客戶登記城市與需求
2. Gavin 確認日期、場地與成本
3. 場次達 BOOKING_READY
4. 開放服務與時段
5. 客戶選擇時段
6. 支付訂金
7. 預約正式成立
8. 發送行前通知

第一版建議：

- 一般城市服務日：服務費 50% 訂金
- 單人指定到訪：確認報價後支付 50% 訂金
- 付款期限：收到通知後 24 小時
- 未付款則時段釋出

## 10. 同行折抵

同行折抵不是常態優惠。

只在以下狀況啟動：

- 差一筆即可成行
- 已成行但仍有空位
- 新需求能補滿延長時段
- 新需求足以加開第二日

建議：

- 差一筆即可成行：雙方各 NT$300～600
- 已成行仍有空位：新客當次折 NT$300，邀請人 NT$300 下次服務金
- 已額滿：不折本期，改候補、下一場或加開評估
- 成功加開第二日：雙方最高各 NT$500
- NT$600 必須由 Gavin 人工確認

## 11. 南部多日行程

台南、高雄可安排：

- 兩天一夜
- 三天兩夜

時段用途：

- 白天：教練課、紫微
- 下午：筋膜整理
- 晚間：運動按摩、塔羅
- 深夜：限量運動按摩

深夜時段：

- 正常時段接近滿額後才開
- 一晚最多一位深夜徒手服務
- 隔天不排過早
- 額外場地與深夜交通須納入試算

多住一晚前必須符合：

> 新增確認營收 ≥ 額外住宿＋場地＋餐食＋交通＋該日目標淨收入

## 12. 外地教練課

目前：

- 台中：可以安排教練課
- 其他外縣市：尚未常態提供

有經驗訓練者可預約單次技術校準。

新手或需要長期改變者，未來規劃：

**BodyFix 混合式教練計畫**

內容：

- 每月一次實體教學
- 四週個人課表
- 每週一次進度回報
- 動作影片回看
- 遠距修正
- 下次城市服務日重新測試

## 13. LINE v0.1

固定入口：

- 查看城市場次
- 登記希望城市
- 單人指定到訪
- 我的登記狀態
- 聯絡 Gavin

固定 Auto-reply 不顯示容易過期的即時人數或差額。

v1.0：

LINE Webhook  
→ Vercel API Route  
→ Supabase  
→ LINE Reply API

Webhook 上線後，關閉重複的 Auto-reply。
