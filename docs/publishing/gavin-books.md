# Gavin Books｜完整出版系列

Gavin Books 是從 Gavin Loam 或其他研究材料中提煉成熟，並經過編輯、驗證與定稿後形成的完整專題作品。

可能包含：

- 數位小書
- 方法手冊
- 專題文章系列
- 可公開下載的工具包

Gavin Books 是出版結果，不是原始資料存放區。

## 與 Gavin Loam 的關係

Gavin Loam 可以保存研究材料、歷史版本、觀察與未完成框架。Gavin Books 只保存已經被提煉、編輯、驗證與定稿的完整作品。

## 與 Public Content 的關係

Public Content 可以是文章、圖卡、影片或公開工具，不一定形成完整系列。Gavin Books 代表更完整、可閱讀、可下載或可出版的專題作品。
